Plateforme de Constitution de Groupes

Application SaaS · Architecture REST MVC

Application d'assistance à la création de groupes de TD/TP pour des promotions de 100+ étudiants. Gestion de contraintes multiples : niveaux académiques, proportions de genre, covoiturages.

Moteur algorithmique

  • Génération manuelle ou automatique des groupes
  • Algorithmes gloutons et force brute en Java pour optimiser les scores d'équilibrage
  • Minimisation du score M+N (écart de moyenne + étudiants sans entreprise)

Architecture technique

  • Infrastructure REST séparant modèle, vue et contrôleur
  • Client lourd Java ↔ serveur PHP via HTTP / JSON
